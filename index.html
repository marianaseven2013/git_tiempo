1 git clone cre una copia de repositorio remoto
2 git status Muestra el estado del repositorio
3 git init Inicializa un nuevo repositorio de Git en el directorio actual.
4 git add Añade archivos al área de preparación (staging area).
5 git commit Guarda los cambios en el repositorio con un mensaje descriptivo.
6 git status Muestra el estado de los archivos en el directorio de trabajo y el área de preparación.
7 git push Envía los cambios confirmados (committed) al repositorio remoto.
